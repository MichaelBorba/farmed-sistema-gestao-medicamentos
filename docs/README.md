# Farmed - Sistema de Gestão de Medicamentos

## Sobre o Projeto

Sistema web desenvolvido para auxiliar instituições de acolhimento no controle de medicamentos, prescrições médicas, renovação de receitas e registro de administração de medicamentos.

O sistema utiliza banco de dados relacional para armazenar informações dos acolhidos, histórico de medicações e alertas de vencimento.

## Tecnologias Utilizadas

* HTML
* CSS
* JavaScript
* Supabase
* SQL

## Funcionalidades

* Cadastro de pacientes
* Controle de medicamentos
* Registro de administração de medicação
* Controle de horários
* Alertas automáticos de vencimento
* Relatórios em PDF
* Importação de dados
* Histórico completo de administração

## Estrutura do Banco de Dados

O banco de dados é composto pelas tabelas:

* pacientes
* registro_medicacao
* calendario

## Objetivo

Automatizar processos que normalmente são realizados em planilhas e documentos manuais, proporcionando maior controle, rastreabilidade e segurança na administração de medicamentos.
